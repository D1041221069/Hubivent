# Cara Penggunaan Aplikasi
## Clone Repository
```
git clone git@github.com:D1041221069/Hubivent.git
```

## Backend Setup
```
cd backend
npm install
```

Sesuaikan file .env dengan pengaturan XAMPP/MySQL kalian, JANGAN LUPA HIDUPKAN XAMPP/MySQL sebelum menjalankan backend.

```
npm run db
npm run db:seeder
npm run start
```

## Mobile Setup
```
cd mobile
npm install
npm run start
```
